# Иконки HOMEd для Home Assistant

## Инструкции
Чтобы добавить эти иконки в Home Assistant, нужно выполнить простые шаги:

1. Добавить этот репозиторий как пользовательский
2. Найти и установить в HACS **Homed Icons**
